# Tela Spotify

Este projeto é uma reprodução da tela do site do Spotify, realizado junto com o professor Jamilton Damasceno no curso **Desenvolvimento Web Completo** da Udemy. Utiliza **HTML** e **CSS** para criar a estrutura e o design.

## Funcionalidades

- Barra de navegação fixa com links para diferentes seções (Premium, Ajuda, Baixar, Inscrever-se, Entrar).
- Carrossel de imagens com chamadas para ação.
- Seção com informações sobre o que o Spotify oferece (músicas, playlists, novos lançamentos).
- Seção de recursos com dicas de como usar o Spotify.
- Galeria de imagens ilustrativas.
- Rodapé com informações sobre a empresa, comunidades e links úteis.

## Tecnologias Utilizadas

- **HTML**: Para a estruturação da página.
- **CSS**: Para estilização e layout, utilizando Bootstrap para facilitar o design responsivo.

## Estrutura do Projeto

- **index.html**: Contém a estrutura principal da página.
- **css/estilo.css**: Define o estilo visual da página.
- **Imagens**: Arquivos de imagens utilizados no design, como logotipo e ilustrações.
